LEARN EASY WEBSITE



En estas tres unidades se desarrolla se implementa de forma pedagogíca el aprendizaje basico para proyectos de diseño web con herramientas de las Tics (Tecnología de la información y de la comunicación) como soporte, también la Universidad Cuenta con un campus virtual el cual posee elementos de aprendizaje interactivo como OVAS, Tutoriales y Vídeos, que se complementan con la aplicación de actividades académicas en línea y la utilización de canales síncronos y asíncronos de comunicación para el desarrollo del proceso que propone el curso. (Sillabus Curso de Diseño Web, 2017)

Se estructura este curso con 2 modulos didacticos: - Estructura pagina web - HTML5 - CSS3

Las tres unidades abordan aspectos de desarrollo , implementación y administración de proyectos de diseño web, para el curso se define a su vez el uso de las Tics como agente mediador en el curso entre Docentes y Estudiantes; para esto la Universidad Cuenta con un campus virtual el cual posee elementos de aprendizaje interactivo como OVAS, Tutoriales y Vídeos, que se complementan con la aplicación de actividades académicas en línea y la utilización de canales síncronos y asíncronos de comunicación para el desarrollo del proceso académico que propone el curso.
